# Gruland

## JSON files (Elixir)

In a terminal run:

```bash
mix deps.get
iex -S mix
```

This open an IEx session. Then run:

```elixir
iex> Builder.run
iex> Npm.find("@aws-sdk/client", "2022-01-01", "2023-01-01")
```

A file "result.json" is created
